# 荔城町第一小学校 – 若い心が花開く場所

広西チワン族自治区桂林市荔浦市の中心に位置する荔城町第一小学校（略称：荔城一小）は、**85年以上**の歴史を持つ公立の名門小学校です。2つのキャンパスを有し、「子供たちの生涯発展の基礎を築く」という理念のもと、数え切れないほどの卒業生を世に送り出してきました。

## 近代的な学習環境
本校のメインキャンパスは 26,640 平方メートルを占め、校舎面積は 22,840 平方メートルを超えます。現在、36 学級、2,451 人の児童と 124 人の高度な専門性を備えた教職員が在籍しています。マルチメディアネットワーク、コンピュータ室、語学室、図書館、理科実験室など、自治区モデル校の基準を満たす設備を完備し、子供たちが知識を探求し、創造性を育むのに最適な環境を提供しています。

## 五育の調和ある発展
当校は「教育の実験化、管理の科学化、研究による学校活性化」という方針を掲げ、徳・知・体・美・労の調和ある発展を重視しています。革新的な授業、課題研究、豊富なクラブ活動を通じて、学力向上だけでなく、創造力、忍耐力、社会貢献の精神も育みます。一人ひとりの個性が輝く教育を実践しています。

## 全国国防教育特色校
**「全国国防教育モデル校」** に認定されており、愛国心と市民としての責務を育むことを大切にしています。児童は国旗護衛隊、軍事訓練、赤いテーマの文化活動などに参加し、規律と創造性を兼ね備えた成長を遂げます。この特色ある教育は市全体で高く評価され、複数回にわたり市の教育現場会議を主催し、広西各地から教育関係者が視察に訪れています。

## 輝かしい実績
これまでに以下の多くの栄誉を受賞しています：
- 広西現代教育技術実験校
- 広西創新教育実験校
- 広西活動課程実験校
- 広西視聴覚教育実験校
- 自治区卓球伝統校
- 桂林市教育研究モデル校
- 桂林市芸術教育モデル校
- 桂林市「文明キャンパス」
- 国際エコスクール（グリーンフラッグ）

これらの賞は、本校の質の高い教育、研究革新、芸術・スポーツ、そして環境保全への取り組みを如実に示しています。

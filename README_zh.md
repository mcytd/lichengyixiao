# 荔城镇第一小学 —— 让每一颗童心绽放光彩

荔城镇第一小学（简称“荔城一小”）位于广西桂林荔浦市城区中心，是一所拥有**85余年**办学历史的优质公立小学。我们拥有两个校区，始终秉持“为学生的终身发展奠基”的信念，培育了一代又一代优秀学子。

## 现代化校园
主校区占地 26640 平方米，校舍面积超过 22840 平方米，教学设施先进齐全。现开设 36 个教学班，在校学生 2451 人，教职工 124 人，均具备高水平专业素养。学校配备多媒体网络、计算机教室、语音室、图书馆、科学实验室等，达到自治区示范性学校标准，为孩子们提供探索知识、启迪智慧的成长空间。

## 全面发展，五育并举
我们坚持“教育实验化、管理科学化、科研兴校”的办学思路，注重德、智、体、美、劳融合发展。通过创新课堂教学、课题研究与丰富的社团活动，不仅提升学业成绩，更培养创造力、抗挫力与社会责任感，让每个孩子都找到自己的闪光点。

## 全国国防教育特色学校
我校荣获 **“全国国防教育示范学校”** 称号，将爱国主义与国防教育融入日常。学生们参与国旗护卫队、军事化操练、红色主题文化活动，在纪律与创意中成长。这一特色教育享誉全市，曾多次承办市级教育现场会，接待来自全区各地的教育同仁观摩交流。

## 荣誉满载，品质见证
近年来，学校先后获得：
- 广西现代教育技术实验学校
- 广西创新教育实验学校
- 广西活动课程实验学校
- 广西电化教育实验学校
- 自治区乒乓球传统项目学校
- 桂林市教育科研示范学校
- 桂林市艺术教育示范学校
- 桂林市“文明校园”
- 国际生态学校绿旗荣誉

这些荣誉充分体现了我校在素质教育、科研创新、艺体特色和生态文明方面的扎实成果。
